## Install

Clone the repo and install dependencies:

```bash
git clone https://github.com/hackyeah-stonly-2025/hackyeah-2025.git
cd hackyeah-2025
npm install
```

## Starting the app

Start the app in the `dev` environment:

```bash
npm start
```
